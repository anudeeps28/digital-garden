# We Learn by Doing, Not by Watching

## Idea
You don't need to know a lot to start — but you need to start a lot to know.

## Definition
Watching, reading, and studying are necessary but insufficient for learning. The knowledge that sticks, the skills that compound, come from actually doing the thing. We often avoid action with the excuse "I need to learn more first," but this inverts the learning process. Action generates feedback; feedback generates understanding; understanding generates better action. The loop requires entry through doing, not knowing.

## Source
Personal observation; consistent with experiential learning theory (Kolb).
