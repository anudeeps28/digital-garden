# Kanban

## Idea
Visualize work as cards moving through stages on a board to make flow visible and limit work-in-progress.

## Definition
Kanban is a workflow management method originating in Toyota's manufacturing system. Work items are represented as cards on a board with columns representing stages (e.g., To Do → In Progress → Done). The key principle: limit Work In Progress (WIP) at each stage to prevent bottlenecks and maintain flow. By making all work visible on one board, teams can identify blockers, balance load, and continuously improve throughput.

In personal productivity, tools like Trello, Notion, or Linear implement Kanban-style boards for task management.

## Source
Toyota Production System (Taiichi Ohno); widely adopted in software development through Agile frameworks.
